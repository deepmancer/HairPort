\paragraph{Baselines.}
We compare with state-of-the-art methods:
\textbf{GAN-based:} HairFastGAN~\cite{hairfastgan}, Barbershop~\cite{barbershop}, Style-Your-Hair~\cite{styleyourhair};
\textbf{CLIP-guided:} HairCLIP~\cite{hairclip}, HairCLIPv2~\cite{hairclipv2};
\textbf{Diffusion-based:} StableHair~\cite{stablehair}, HairFusion~\cite{hairfusion}.

\paragraph{Evaluation Metrics.}
We evaluate three aspects:
\begin{itemize}[noitemsep,leftmargin=*]
    \item \textbf{Hair Transfer Quality:} $\dinoHair$ and $\clipHair$ measure cosine similarity between DINOv2/CLIP features of reference and generated hair regions.
    \item \textbf{Identity Preservation:} IDS using ArcFace~\cite{arcface} embeddings, CLIP-I for overall similarity.
    \item \textbf{Image Quality:} FID~\cite{fid}, LPIPS~\cite{lpips}, SSIM, DreamSim~\cite{dreamsim}.
\end{itemize}
