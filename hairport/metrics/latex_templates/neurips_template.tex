\paragraph{Datasets.}

\paragraph{Baselines.}
We compare against state-of-the-art hair transfer methods:
HairFastGAN~\cite{hairfastgan},
StableHair~\cite{stablehair},
HairCLIP~\cite{hairclip},
HairCLIPv2~\cite{hairclipv2},
HairFusion~\cite{hairfusion},
Barbershop~\cite{barbershop}, and
Style-Your-Hair~\cite{styleyourhair}.

\paragraph{Metrics.}
We evaluate using:
\begin{itemize}[noitemsep,topsep=0pt]
    \item \textbf{Hair Transfer Quality:} \dinoHair{} and \clipHair{} measure similarity between reference and generated hair regions using DINOv2 and CLIP features.
    \item \textbf{Identity Preservation:} IDS (Identity Similarity) using InsightFace embeddings, and CLIP-I for overall image similarity.
    \item \textbf{Image Quality:} FID, LPIPS, SSIM, and DreamSim for perceptual quality assessment.
\end{itemize}
